# Prova Remota - Qualidade e Teste de Software

Este repositório contém a prova remota em arquivo único para publicação no GitHub Pages.

## Como publicar

1. Crie um repositório no GitHub.
2. Envie os arquivos `index.html` e `.nojekyll` para a raiz do repositório.
3. Vá em Settings > Pages.
4. Em Build and deployment, selecione:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Clique em Save.
6. Aguarde a publicação e copie o link gerado pelo GitHub Pages.

## Observação

O arquivo `index.html` contém todo o WebApp embutido: HTML, CSS, JavaScript e banco de questões.
